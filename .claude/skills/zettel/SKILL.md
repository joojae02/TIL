# 제텔카스텐 노트 생성

## 개요

하나의 개념 = 하나의 노트 원칙으로 `Resources/Zettelkasten/`에 원자적 노트를 생성합니다.

## 절차

1. **주제 파악**: 사용자가 정리하려는 개념 확인
2. **기존 노트 검색**: `Resources/Zettelkasten/`에서 관련 노트가 있는지 확인
3. **노트 생성**: 아래 템플릿으로 새 노트 작성
4. **링크 제안**: 연결할 수 있는 기존 노트들을 `[[링크]]` 형태로 제안

## 노트 템플릿

```markdown
# {개념 이름}

## 정의

{한 문장으로 핵심 정의}

## 설명

{상세 설명}

## 예시

{코드나 실제 사례}

## 관련 개념

- [[관련노트1]]
- [[관련노트2]]

---
생성일: {YYYY-MM-DD}
```

## 파일명 규칙

- 짧고 명확하게 (예: `HTTP 상태코드 4xx.md`, `Redis TTL.md`)
- 제목만 봐도 내용을 알 수 있게
- 공백 허용, 특수문자는 최소화

## 예시

사용자: "Redis의 TTL에 대해 정리해줘"

→ `Resources/Zettelkasten/Redis TTL.md` 생성
→ 관련 노트 `[[Redis]]`, `[[캐시 만료 전략]]` 링크 제안
