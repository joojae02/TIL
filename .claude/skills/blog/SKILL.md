# 블로그 글 초안 생성

## 개요

`Projects/개발블로그/` 또는 `Projects/취미블로그/`에 새 블로그 글 초안을 생성합니다.

## 절차

1. **블로그 유형 확인**: 개발 블로그인지 취미 블로그인지 확인 (불명확하면 질문)
2. **주제 파악**: 어떤 내용을 쓸 것인지 확인
3. **관련 Zettelkasten 노트 검색**: `Resources/Zettelkasten/`에서 참고할 노트 찾기
4. **초안 생성**: 아래 템플릿으로 초안 작성
5. **링크 삽입**: 관련 제텔카스텐 노트들을 본문에 `[[링크]]`로 연결

## 블로그 템플릿

```markdown
---
status: draft
created: {YYYY-MM-DD}
tags: []
---

# {제목}

## 들어가며

{글의 배경이나 동기}

## 본문

{핵심 내용}

## 마치며

{요약 및 마무리}

---

## 참고 자료

- [[관련노트1]]
- [[관련노트2]]
```

## 파일명 규칙

- 형식: `초안-{제목}.md`
- 예시: `초안-Redis 캐시 전략 정리.md`

## 폴더 구조

```
Projects/
├── 개발블로그/     ← 기술, 개발 관련 글
│   └── 초안-제목.md
└── 취미블로그/     ← 취미, 일상 관련 글
    └── 초안-제목.md
```

## 예시

사용자: "Redis 캐시 전략에 대해 블로그 글 써줘"

1. 유형: 개발블로그
2. `Resources/Zettelkasten/`에서 Redis 관련 노트 검색
3. `Projects/개발블로그/초안-Redis 캐시 전략 정리.md` 생성
4. 관련 노트들 `[[Redis TTL]]`, `[[캐시 무효화]]` 본문에 링크
