
## 추천 시스템이란??

 ![](https://i.imgur.com/XYltj8v.png)

추천시스템이란 "여러 후보 가운데 가치 있는 것을 선정해서 의사 결정을 지원하는 시스템"이라 설명할 수 있다.

얼마나 중요한가??

- Netflix : 대여되는 영화의 2/3가 추천으로부터 발생
- Google News : 38% 이상의 조회가 추천에 의해 발생
- Amazon : 판매의 35% 가 추천으로부터 발생
- Netflix Prize (~2009) Netflix에서 주관하는 경연대회로, 영화 선호도를 가장 잘 예측하는 협업 필터링 알고리즘에 수상 (US$1,000,000)

### 추천시스템의 구성  및 종류

크게 

1. 입력 (데이터 입력)
2. 프로세스 (추천 설계)
3. 출력 (추천 결과 출력)

으로 이뤄진다

그리고 추천 시스템은
크게

1. 내용 기반 필링
2. 협조 필터링

으로 나눠진다.

### 데이터를 이용한 추천?

1. 개요 추천

판매 순위 와 같은 최신순, 인기순, 낮은 가격순 과 같은
모든 사용자에게 동일한 데이터가 제공되는 추천을 말한다.

2. 연관 아이템 추천

보통 아이템 페이지 하단에 비슷한 아이템을 추천해준다
이 경우 아이템 간의 유사도를 기반으로 추천해준다.

다만.. 
해리포터 문제라 불리는 문제가 있다.
특정 시기에 대부분의 사람들이 해리포터 서적을 다른 아이템과 구매해
모든 아이템의 추천 상품으로 해리포터 책이 추천되는?? 

3. 개인화 추천

보통 일반적으로 알고 있는 추천이다.
컨텐츠를 기반 혹은 인터렉션 데이터를 활용한다.

##  검색 vs 추천

검색
- 키워드 존재
- 키워드 기반 의도 추측
- 능동적

추천 시스템
- 입력 없음
- 사용자 데이터 혹은 행동으로 추측
- 수동적

### 내용 기반

사용자가 선호하는 정보 기반으로 아이템을 계산한다.
예를들어 A사용자는 미스터리라는 장르를 좋아한다면 미스터리 관련 책을 추천해주준다.

### 협조 필터링

예를 들어 A 유저가 한 이슈에 관해서 B 유저와 같은 의견을 갖는다면
다른 이슈에 대해서도 비슷한 의견을 가질 확률이 높을 것을 전제로 한다.

크게

- Memory-based
- Model-based
- Hybrid
로 구분된다

또

- User-Based
- Item-Based
로 나눠지는데
- 사용자 기반 : 비슷한 고객들이 ~한 제품을 구매했다.
- 아이템 기반 : ~ 상품을 구매한 고객들은 다음 상품도 구매했다.
